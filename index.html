A function f from set A to set B assigns to each element x in A exactly one element f(x) in B. The domain is A. The range is {f(x) : x ∈ A} ⊆ B.
A curve in the xy-plane is the graph of a function iff every vertical line intersects the curve at most once.

Real Numbers — Building from Axioms

Stage II builds all of calculus on three sets of axioms: field axioms (arithmetic), order axioms (inequalities), and the completeness axiom (no gaps). Every theorem in ultimately rests on these foundations.

Closure: a+b∈ℝ. Commutativity: a+b=b+a. Associativity: (a+b)+c=a+(b+c). Zero: ∃0 s.t. a+0=a. Negatives: ∃(−a) s.t. a+(−a)=0.
Closure: ab∈ℝ. Commutativity: ab=ba. Associativity: (ab)c=a(bc). Unity: ∃1≠0 s.t. 1·a=a. Reciprocal: if a≠0, ∃a⁻¹ s.t. a·a⁻¹=1. Distributivity: a(b+c)=ab+ac.

Linear Spaces — The Unifying Framework

A linear space (vector space) is any collection of objects that can be added and scaled. By proving theorems about linear spaces in general, we get results valid for ℝⁿ, function spaces, matrices, polynomials, and infinite-dimensional spaces simultaneously.

A linear space V over ℝ is a non-empty set with addition x+y and scalar multiplication cx satisfying:
(A1) x+y∈V  (A2) x+y=y+x  (A3) (x+y)+z=x+(y+z)
(A4) ∃0: x+0=x  (A5) ∃(−x): x+(−x)=0
(M1) cx∈V  (M2) c(x+y)=cx+cy  (M3) (c+d)x=cx+dx
(M4) c(dx)=(cd)x  (M5) 1·x=x
From the 10 axioms alone, we can prove: 0·x=0, c·0=0, (−1)x=−x, and if cx=0 then c=0 or x=0. These are theorems, not assumptions!

📖Core Concepts
A function f: A→B assigns to each x∈A exactly one value f(x)∈B. The domain is the set of valid inputs; the range is the set of actual outputs. Functions can be given by formulas, tables, graphs, or verbal descriptions.

Composition (f∘g)(x)=f(g(x)): apply g first, then f. Domain of f∘g = {x in domain of g : g(x) in domain of f}. Note: f∘g ≠ g∘f in general.

Inverse functions: A function f has an inverse f−1 iff it is one-to-one (injective): f(x1)=f(x2) implies x1=x2. Then f−1(f(x))=x and f(f−1(y))=y. Graph of f−1 = reflection of graph of f over y=x.

Transformations: y=f(x−h) shifts right h; y=f(x)+k shifts up k; y=af(x) scales vertically by a; y=f(bx) changes period (horizontal scaling).
VERTICAL LINE TESTA curve represents a function iff every vertical line intersects it in at most one point.
HORIZONTAL LINE TESTA function has an inverse iff every horizontal line intersects its graph at most once. Equivalently, f is strictly monotone.

Dot product: u·v=u1v1+⋯+unvn=|u||v|cosθ. Perpendicular iff u·v=0. Projection of u onto v: projv(u)=(u·v/|v|²)v.

📖Core Concepts
The real number system ℝ is built from three axiom groups. Field axioms (A1–A10) govern arithmetic: commutativity, associativity, distributivity, existence of 0, 1, negatives, and reciprocals. Order axioms (B1–B5) establish the ordering: trichotomy, closure of positives under addition and multiplication. Archimedean property: For any x∈ℝ, ∃n∈ℕ with n>x. For any ε>0, ∃n with 1/n<ε. No infinitely large or infinitely small nonzero elements.

This IS the integral: the supremum of lower sums equals the infimum of upper sums — the completeness axiom guarantees they are equal.
COMPLETENESS AXIOMEvery nonempty S ⊆ ℝ bounded above has sup S ∈ ℝ. Guarantees: Cauchy sequences converge, continuous functions attain bounds, and integrals of continuous functions exist.
DENSITY OF ℚBetween any two reals a < b, there exists a rational r with a < r < b. Also an irrational. Both ℚ and ⅈℚ are dense in ℝ.

📖Integration First — The Key Innovation
Stage II defines the integral before limits and derivatives — historically correct and logically sound. A step function s on [a,b] is piecewise constant on some partition. Its integral is purely algebraic: ∫s = sum of (constant value) × (subinterval width). No limits needed.

    The Fundamental Theorem of Calculus
    ▶
  

  
FTC Part IFTC Part IIAntiderivativesSubstitutionIntegration by Parts
📖The Central Result
The FTC forges the connection between integration and differentiation — discovered independently by Newton and Leibniz.

FTC Part I (differentiation undoes integration): If f is continuous and A(x)=∫axf(t)dt, then A'(x)=f(x). The area function is an antiderivative of the integrand.

FTC Part II (integration via antiderivatives): If G'=f on [a,b], then ∫abf(x)dx = G(b)−G(a). To evaluate a definite integral: find any antiderivative, evaluate at endpoints.

Zero Derivative Theorem: If F'(x)≡0 on (a,b), then F is constant. Any two antiderivatives differ by a constant C.

Substitution (u-substitution): ∫f(g(x))g'(x)dx = ∫f(u)du. Reverse of chain rule.

Integration by parts: ∫u dv = uv − ∫v du. Reverse of product rule. LIATE: Log, Inverse trig, Algebraic, Trig, Exponential.
FTC PART IIf f is integrable on [a,b] and A(x)=∫axf(t)dt, then A is continuous. If f is continuous at an interior point x, then A'(x)=f(x).
FTC PART IIIf f is continuous on [a,b] and G'=f, then ∫abf(x)dx = G(b)−G(a). Formula unifies all computational calculus.
FTC I: [A(x+h)−A(x)]/h = (1/h)∫xx+hf(t)dt. By continuity of f at x, for any ε>0 choose δ so |f(t)−f(x)|<ε when |t−x|<δ. Then |(1/h)∫xx+hf(t)dt − f(x)| < ε.

📖Core Concepts
Chebyshev polynomials: Tn(x)=cos(n·arccos x) on [−1,1]. Recurrence: T0=1, T1=x, Tn+1=2xTn−Tn−1. Roots: xk=cos((2k−1)π/(2n)), k=1,...,n.

Minimal property: Among all monic degree-n polynomials, (1/2n−1)Tn has the SMALLEST sup-norm on [−1,1]. This min-max principle also appears in adversarial ML and game theory!

Runge phenomenon: Equal-spacing interpolation nodes cause wild oscillations near endpoints for high degrees. Chebyshev nodes (clustering near endpoints) eliminate this — optimal nodes for polynomial interpolation.

Euler-Maclaurin formula: ∑k=1nf(k) = ∫1nf(x)dx + (f(1)+f(n))/2 + ∑B2j/(2j)!·(f(2j−1)(n)−f(2j−1)(1)) + R. Bridges discrete sums and continuous integrals. Gives Stirling's formula ln(n!)≈n ln n−n.
SIMPSON'S RULE ERROR BOUND|Error| ≤ (b−a)5/(180n4)·max|f(4)|. Error O(h4) vs O(h2) for trapezoid. Simpson is exactly correct for cubics (f(4)=0).
INTERPOLATION ERRORf(x)−p(x) = f(n+1)(ξ)/(n+1)!·∏i(x−xi). Minimised by Chebyshev nodes: max|∏(x−xi)| = (b−a)n+1/22n+1 — optimal.
